# Data Science Projects and Case Studies

This repository is a curated set of data science projects and case studies covering exploratory analysis, machine learning, and applied modeling. Most work is in Jupyter notebooks and is organized by project folder.

- CIFAR100 CNN and Logistic Regression Project
- Credit Card Fraud Capstone

- Investigating Netflix movies project

## Projects

### 1) CIFAR100 CNN and Logistic Regression Project
**Folder:** `CIFAR100_CNN_and_Logistic_Regression_Project/`

**Goal**
- Compare a deep learning approach (CNN) vs a simpler baseline (logistic regression) for image classification on CIFAR-100.

**What I did**
- Data preparation and train/validation/test split
- Baseline modeling with logistic regression
- CNN training and evaluation
- Model comparison using consistent evaluation metrics

**Outputs**
- Training and evaluation notebooks
- Metrics and plots (accuracy and loss curves if included in the notebooks)

---

### 2) Credit Card Fraud Capstone
**Folder:** `Credit card Fraud capstone/`

**Goal**
- Build and evaluate classification models to detect fraudulent transactions.

**What I did**
- Exploratory analysis and data cleaning
- Handling class imbalance (example approaches: class weights, resampling, threshold tuning)
- Model training and evaluation with metrics suitable for imbalanced data (precision, recall, F1, ROC-AUC, PR-AUC)
- Comparison of multiple models and selection based on business objective (reduce false negatives vs false positives)

**Outputs**
- Notebooks with model training and evaluation
- Summary of findings and recommended model approach
