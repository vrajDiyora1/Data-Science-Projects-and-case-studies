# Data Science Projects and Case Studies

A collection of hands-on data science projects (EDA, machine learning, and case studies) using Python and Jupyter Notebooks.

## Projects in this repo
- **CIFAR100 CNN and Logistic Regression Project** (`CIFAR100_CNN_and_Logistic_Regression_Project/`)
  - Image classification experiments with deep learning and baseline models.

- **Credit Card Fraud Capstone** (`Credit card Fraud capstone/`)
  - Fraud detection focused on classification, evaluation, and model comparison.

- **Mountain Resort Pricing Strategy Optimization** (`Mountain Resort pricing strategy optimization project/`)
  - Pricing and optimization case study using data-driven analysis.

- **London Housing Case Study** (`London Housing Case Study Overview Tier_3.ipynb`)
  - Notebook-based case study on housing data analysis.

- **Investigating Netflix Movies Project** (`investigating netflix movies Project/`)
  - EDA-focused project exploring trends and patterns in movie data.

- **Fuel Economy tidyverse project** (`Fuel_Economy_tidyverse_library_project/`)
  - Fuel economy analysis (includes work aligned with the tidyverse workflow).

- **Case studies** (`Case studies/`)
  - Additional case studies and practice notebooks.

- **Mini Project** (`Mini Project/`)
  - Smaller experiments and learning projects.
