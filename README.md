# Data Science Projects and Case Studies

This repository contains my data science projects and case studies focused on machine learning, experimentation, and data-driven problem solving using Python and SQL.

## What you will find here
- Exploratory data analysis (EDA) and feature engineering
- Supervised learning: classification and regression
- Model evaluation and interpretation
- End-to-end case studies with clear problem statements and results

## Skills and tools
Python | SQL | Jupyter | Git  
pandas | numpy | scikit-learn | matplotlib | seaborn

## Repository structure
- `EDA/` : notebooks for data exploration and cleaning
- `Models/` : training pipelines, model comparisons, evaluation
- `Features/` : feature engineering and preprocessing
- `Reports/` : summaries of approach, metrics, and conclusions
- `Data/` : sample data or links to data sources

## How to use
1. Open any project folder and read its `README.md` first.
2. Run notebooks in order (EDA → modeling → evaluation).
3. Each project includes dataset info, steps, and results.
